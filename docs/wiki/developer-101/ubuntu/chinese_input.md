# Ubuntu 中文输入法
[中州韵输入法](http://rime.im/)

#### 软件安装

	sudo apt-get install language-pack-zh-hans fcitx-rime
	
#### 设置输入法框架

	im-config -n fcitx
	！！！重启系统
	
#### 设置输入法

	fcitx-config-gtk3

**设置简体中文**

* CTRL + Space(切换到 rime 输入法)
* CTRL + ~ (输入方案选择)
